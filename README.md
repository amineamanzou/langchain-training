# LangChain Training

Public LangChain training notes, organized as a Hugo documentation site.

## Purpose

This repository tracks theory notes, practice plans, and training progress in public.
The public surface is the Hugo site in `site/`.

Practice repositories, notebooks, generated data, and local secrets stay outside Git in
ignored local folders.

## Structure

- `site/`: Hugo documentation source.
- `site/content/notes/`: Zettelkasten-style evergreen notes.
- `site/content/trainings/`: training session pages and practice checklists.
- `practice/`: ignored local workspace for cloned training repositories and notebooks.

## Local Practice

Clone course repositories into `practice/` and follow their notebook instructions there.
That work is intentionally local-only and is not pushed to this repository.
